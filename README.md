# Crescendo

**Crescendo** is a community-driven music platform for discovering, rating, and sharing your favorite songs and albums.

## Features

* Discover artists, albums, and songs.
* Search for songs and users.
* Rate songs from 1 to 5 stars.
* Write and view reviews.
* Follow users and artists.
* View personalized activity through the home feed.
* Save songs and albums to **Listen Later**.
* Compare music tastes with other users using **Taste Match**.
* Add new songs, albums, and artists as a verified user.

## Requirements

Before running the application, install:

* JDK 21 or later
* Apache Maven
* MySQL Server
* Git

## Architecture

Crescendo uses an MVC-inspired layered structure:

* **Model:** Represents users, artists, albums, songs, reviews, and tags.
* **View:** Contains the JavaFX UI components and application pages.
* **Controller:** Coordinates actions between the UI, models, and database.
* **Database:** Handles MySQL database initialization, queries, authentication, and persistent data.
